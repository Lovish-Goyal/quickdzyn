# QuickDzyn

QuickDzyn is a full-stack template marketplace with a Next.js client and an Express + MongoDB server.

## Structure
- `client/` – Next.js (App Router) + Tailwind + Framer Motion
- `server/` – Express + MongoDB API

## Quick Start

```bash
cd server
npm install
npm run dev
```

```bash
cd ../client
npm install
npm run dev
```

See `setup.md` for full environment setup and deployment instructions.
